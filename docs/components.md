# Component Library

## Shared Components

- `Card` - Standard container with shadow and border
- `SectionHeading` - Section header with optional eyebrow text
- `StatCard` - KPI display for dashboards
- `Button`, `Input`, `Select`, `TextArea` - Form controls
- `Badge` - Status labels
- `LoadingSkeleton` - Loading state
- `TopNav`, `Footer`, `ThemeToggle`, `NotificationBell` - Layout elements
